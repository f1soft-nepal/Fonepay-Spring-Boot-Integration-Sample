## Dependency (Optional):

You can use following fonepay java library.

https://github.com/fonepay/Fonepay-Java

   Maven:
    
        <dependency>
            <groupId>com.fonepay</groupId>
            <artifactId>payment</artifactId>
            <version>0.9</version>
        </dependency>
        
   Gradle:
     
     compile group: 'com.fonepay', name: 'payment', version: '0.9'
